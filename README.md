# Python Data Analysis Portfolio

A collection of data analysis projects utilizing core Python libraries—**NumPy**, **Pandas**, and **Matplotlib**—to extract insights, clean datasets, and visualize trends from real-world data sources.

---

## Repository Structure

* 📓 `Project1_Weather_Data_Analysis_Using_NumPy,_Pandas,_and_Matplotlib.ipynb`
* 📓 `Project2_Movie_Ratings_Analysis_Using_NumPy,_Pandas,_and_Matplotlib.ipynb`

---

## Project Directory

### 🌦️ Project 1: Weather Data Analysis
This project focuses on processing time-series environmental data to analyze weather patterns, temperature anomalies, and precipitation distributions.

* **Key Objectives:**
  * Clean missing data records and filter weather anomalies using **Pandas**.
  * Perform fast array computations and calculate structural averages using **NumPy**.
  * Generate line graphs, histograms, and scatter plots using **Matplotlib** to visualize seasonal temperature shifts.
* **Core Skills Featured:** Time-series indexing, boolean masking, conditional data filtering, and trend line plotting.

### 🎬 Project 2: Movie Ratings Analysis
An exploratory data analysis (EDA) project evaluating entertainment industry metrics, user demographics, and historical movie tracking scores to uncover audience preferences.

* **Key Objectives:**
  * Merge multi-table relational data structures (Users, Movies, and Ratings tables) seamlessly.
  * Aggregate, group, and rank top-rated titles using complex criteria with **Pandas**.
  * Handle dense numerical arrays to compute descriptive statistical parameters via **NumPy**.
  * Create distributions of ratings and trend visualizations using **Matplotlib**.
* **Core Skills Featured:** Multi-table joins (`merge`/`join`), data aggregation (`groupby`), pivot tables, and distribution charts.
